# 💻 Sabum Terminal Portfolio (변사범 포트폴리오)

변사범 (Sabum Byun) - Creative Director & Brand Designer 의 이력과 포트폴리오 프로젝트를 담은 인터랙티브 터미널 컨셉 웹사이트입니다.

🔗 **Live Demo:** [https://minnnxk.github.io/portfolio/](https://minnnxk.github.io/portfolio/)

## ✨ 주요 기능 및 특징 (Features)

- **레트로 터미널 컨셉 UI**: BIOS 부팅 시퀀스부터 프롬프트 명령어 입력까지 실제 운영체제 터미널을 다루는 듯한 몰입감을 선사합니다.
- **인터랙티브 명령어 (CLI)**: `help`, `whoami`, `career`, `projects` 등의 터미널 명령어를 직접 치면서 프로필을 탐색할 수 있습니다.
- **글리치(Glitch) 및 스크램블 효과**: 탭 전환, 마우스 오버 시 텍스트 해독 및 RGB 분리 노이즈 애니메이션을 적용해 레트로 해커/개발자 감성을 극대화했습니다.
- **자연스러운 다국어 토글 (KO/EN)**: 우측 하단 상태바(Status bar) 버튼 클릭 시, 전체 문서가 글리치 효과와 함께 영문으로 실시간 통역(변환)됩니다.

## 🛠 기술 스택 (Tech Stack)

- **HTML5** 
- **Vanilla CSS3** (CSS Variables, Custom Keyframe Animations, 글리치 애니메이션)
- **Vanilla JavaScript** (ES6+, Text Scrambler 구조 설계)
- *No External Libraries & Frameworks* (외부 라이브러리 없이 전부 직접 구현되었습니다)
